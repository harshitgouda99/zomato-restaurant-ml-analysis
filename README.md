# zomato-restaurant-ml-analysis
Machine learning project analyzing Zomato restaurant data using EDA, NLP-based sentiment analysis, and ML models to understand factors affecting restaurant ratings.

This project performs a comprehensive analysis of the Zomato restaurant dataset to understand the factors influencing restaurant ratings and customer satisfaction. The project includes data cleaning, exploratory data analysis, feature engineering, sentiment analysis on customer reviews, and machine learning model implementation.

Various visualizations were created to identify patterns in restaurant cost, review sentiment, and ratings. Multiple machine learning models such as K-Means Clustering, Random Forest Regressor, and Decision Tree Regressor were implemented and evaluated using metrics like R² Score and RMSE.

The final model was optimized using hyperparameter tuning and saved for deployment using joblib. The results provide useful insights into customer preferences and restaurant performance, which can help improve recommendation systems and business decision-making.
